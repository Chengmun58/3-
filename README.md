# 매일, 한국어 — 개인 AI 한국어 아카이브

브라우저에서 바로 실행되는 개인용 한국어 몰입 학습 웹사이트입니다.

## 실행

`index.html`을 열거나, 폴더에서 다음 명령을 실행합니다.

```bash
python3 -m http.server 8080
```

그다음 `http://localhost:8080`에 접속합니다.

## 현재 기능

- 날짜에 따라 바뀌는 오늘의 학습 콘텐츠
- 말하기 메모와 글쓰기 저장
- 간단한 글쓰기 길이·연결 표현 점검
- 일일 체크리스트와 완료율
- 표현 저장
- 기록 검색, 유형/월 필터, 정렬
- 기록 상세 보기와 삭제
- 연속 기록, 표현 수, 글 수 통계
- JSON 백업 내보내기
- Supabase 로그인 및 클라우드 동기화
- 미로그인 시 localStorage 기반 저장

## 클라우드 연결

Supabase 프로젝트 `ai-korean-portfolio`에 연결되어 있습니다. 로그인하면 개인 기록이 클라우드에 동기화됩니다.
